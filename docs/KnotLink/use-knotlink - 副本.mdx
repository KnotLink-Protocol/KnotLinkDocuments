# 使用 KnotLink

## 1 信号发送者(SignalSender)

信号发送者（SignalSender）负责生成和发送信号。你可以将其想象为一个广播员，他通过麦克风向所有人广播消息。

#### 配置 SignalSender

在发送信号之前，你需要配置 SignalSender 的 **APPID** 和 **SignalID**。这些是用于标识信号的唯一标识符。

<MultiLangCodeTabs
  qtcpp={``}
  pyqt5={`# 配置 APPID 和 SignalID
signalSender.setConfig(appID="0x00100000", signalID="0x00000010")`}
  python={`# 配置 APPID 和 SignalID
signalSender.setConfig(appID="0x00100000", signalID="0x00000010")`}
  cpp={`signalSender->setConfig("0x00100000","0x00000010");`}
  java={``}
  js={``}
  lua={``}
/>

#### 发送信号

一旦配置完成，你就可以发送信号了。发送信号时，你需要提供一个字符串格式的数据。

<MultiLangCodeTabs
  qtcpp={``}
  pyqt5={`# 发送信号

  python={`# 发送信号
signalSender.emitt(data="Hello, World!")`}
  cpp={`signalSender->emitt("Hello, World!");`}

#### 定义接收数据的回调函数（Qt：槽函数）

当 SignalSender 发送信号时，SignalSubscriber 需要一个回调函数来处理接收到的数据。

<MultiLangCodeTabs
  qtcpp={``}
  pyqt5={`def on_received_data(data: str):
    print(f"接收到字符串数据：{data}")`}
  python={`def on_received_data(data: str):
    print(f"接收到字符串数据：{data}")`}
  cpp={`void onDataReceived(const std::string& data) {
	  std::cout << "Received data: " << data << std::endl;
}`}
  java={``}
  js={``}
  lua={``}
/>

#### 连接回调函数（Qt：槽函数）

将回调函数绑定到 SignalSubscriber 的 `received_data` 事件。

<MultiLangCodeTabs
  qtcpp={``}
  pyqt5={`# 连接回调函数
signalSubscriber.received_data.connect(on_received_data)`}
  python={``}
  cpp={`subscriber->setOnDataReceivedCallback(onDataReceived);`}
  java={``}
  js={``}
  lua={``}
/>
